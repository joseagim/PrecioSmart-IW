PrecioSmart - Preguntas frecuentes

Preguntas frecuentes (FAQ)

Esta página recoge las dudas más habituales sobre el funcionamiento de la aplicación PrecioSmart.

1. ¿Cómo puedo buscar o comparar un producto?

Usa la página de Búsqueda. Puedes introducir el nombre manualmente o usar la opción de escaneo de código de barras. La aplicación mostrará productos similares y te permitirá filtrarlos por precio, peso, grasas, azúcar u otros criterios.

2. ¿Cómo creo y gestiono un carrito?

En la sección Carrito puedes crear un nuevo carrito, añadir productos y ver el precio total. También se muestra una comparación de precios entre distintos supermercados para los productos del carrito.

3. ¿Necesito una cuenta para usar la app?

No todas las funciones requieren cuenta, pero para crear carritos, enviar solicitudes, recibir notificaciones y personalizar el perfil debes registrarte e iniciar sesión.

4. ¿Cómo cambio mi foto de perfil?

Desde tu perfil hay una opción para seleccionar un archivo de imagen (jpg/png). Tras seleccionar la imagen verás una previsualización y podrás confirmar para usarla como avatar. La imagen se sube al servidor y se usa en el menú y en tu perfil.

5. ¿Qué responsabilidades tiene un administrador?

Los administradores tienen acceso al panel de administración para gestionar usuarios, productos y solicitudes. Pueden validar o rechazar peticiones de nuevos productos y editar entradas del catálogo.

6. ¿Qué es la funcionalidad de solicitudes (requests)?

Si no encuentras un producto puedes enviar una solicitud. El equipo (o los admins) puede revisar la solicitud, añadir el producto al catálogo y notificarte cuando esté disponible.

7. ¿Cómo funcionan los mensajes y notificaciones?

La aplicación usa WebSocket para notificaciones en tiempo real. Puedes enviar mensajes a otros usuarios desde su perfil y recibir avisos (por ejemplo, cuando un admin responde a tu solicitud o cuando hay cambios relevantes).

8. ¿Puedo subir imágenes de productos?

Sí. Al crear o editar productos puedes subir imágenes. El sistema acepta formatos comunes (jpg/png); comprueba el tamaño y la resolución para garantizar una carga correcta.

9. ¿Cómo se protege mi información?

Las operaciones protegidas utilizan tokens CSRF y control de sesiones. Los datos sensibles (contraseñas) se almacenan de forma segura en el servidor. Consulta la política de privacidad para más detalles.

10. ¿Qué hago si encuentro un error o falta un producto?

Usa la sección de Solicitudes para pedir productos nuevos o contacta con el equipo mediante el formulario de contacto (o el correo que aparece en la página principal). Adjunta detalles y capturas si es posible.

11. ¿Cómo funciona la subida de avatar en el perfil (detalles técnicos)?

El selector de archivo genera una previsualización en el cliente (FileReader). Al confirmar, la imagen se envía al servidor como multipart/form-data y el servidor actualiza la imagen asociada al usuario. La caché se evita añadiendo un parámetro único (cache buster) a la URL.

12. ¿Puedo eliminar o cambiar información de mi cuenta?

Desde el área de perfil puedes modificar datos personales (dentro de las opciones disponibles). Para eliminación completa de cuenta contacta con el soporte indicando tu usuario.

13. ¿Dónde puedo ver la lista completa de autores y créditos?

En la página Authors (en el pie o barra de navegación) encontrarás los autores y sus fotos.

14. ¿Hay límites en el tamaño de las imágenes?

Se recomiendan imágenes optimizadas. El servidor puede rechazar ficheros muy grandes; si tienes problemas, reduce la resolución antes de subir.

15. ¿Cómo contribuyo al proyecto o consulto el código?

El proyecto dispone de un repositorio con el código fuente. Consulta el README y el LICENSE en el repositorio para instrucciones sobre cómo colaborar.

Si tu duda no aparece aquí, ponte en contacto usando el formulario de contacto o abre una incidencia describiendo el problema.
